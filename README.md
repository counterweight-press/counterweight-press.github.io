# Counterweight

**History has more than one narrator.**

Anonymous, sourced counter-narratives to the stories nations tell themselves. No bylines. No authors. Just evidence and argument.

Built with Jekyll. Hosted on GitHub Pages. Comments via [Staticman](https://staticman.net/).

---

## Reading

Articles are published at the site root. Each piece includes sourced footnotes, a colophon explaining editorial framing, and a collapsible TL;DR.

RSS feed available at `/feed.xml`.

---

## Submitting Comments

Comments are moderated. When you submit a comment, it creates a pull request. Once approved, it appears on the article.

All comments are public. Use whatever name you like.
